# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter scaffolding template that generates production-ready Django REST Framework APIs with batteries included. The next milestone is completing the modernization effort tracked in `.daemon/specs/cookiecutter-django-rest-modernization/` — ensuring the template targets Python 3.13+, Django 5.0+, and PostgreSQL 16.4+ with all dependencies current and CI passing cleanly. Daemon should focus on working through open modernization tasks, keeping dependencies up to date, and improving the generated project's defaults to reflect current best practices.
