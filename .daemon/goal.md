# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter template that scaffolds production-ready Django REST Framework APIs with authentication, user management, Docker, and CI out of the box. The next milestone is modernizing the template to use Python 3.13+, Django 5.0+, and PostgreSQL 16.4+, ensuring all generated project dependencies and patterns reflect current best practices. Daemon should focus on dependency updates, modernizing the generated project structure, improving test coverage of the template itself, and keeping CI green.
