# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter template that scaffolds production-ready Django REST Framework APIs with authentication, user management, Docker, and CI pre-configured. The next milestone is completing the modernization effort: updating the template to Python 3.13+, Django 5.0+, and PostgreSQL 16.4+, ensuring all generated project dependencies, configs, and CI workflows reflect current best practices. Daemon should focus on executing the tasks outlined in `.daemon/specs/` — particularly dependency updates, config modernization, and keeping the generated project's test suite green.
