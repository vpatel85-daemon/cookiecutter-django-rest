# Goal

We are building and maintaining cookiecutter-django-rest, a scaffolding template that lets developers spin up production-ready Django REST Framework APIs in seconds. The next milestone is completing the modernization effort — ensuring the template targets Python 3.13+, Django 5.0+, and PostgreSQL 16.4+, with updated dependencies, CI, and generated project structure reflecting current best practices. Daemon should focus on executing the open modernization specs in `.daemon/specs/`, resolving outdated patterns in the template files, and keeping dependencies current via the pyup workflow.
