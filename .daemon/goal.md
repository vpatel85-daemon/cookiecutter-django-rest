# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter template that scaffolds production-ready Django REST Framework APIs with auth, user accounts, Docker, and CI out of the box. The next milestone is completing the modernization specs — upgrading the template to Python 3.13, Django 5.0+, and PostgreSQL 16.4+, and ensuring all generated project dependencies and CI workflows reflect current best practices. Daemon should focus on working through the open tasks in `.daemon/specs/` (modernization, dependency updates) and keeping the template's generated output clean, tested, and fully functional.
