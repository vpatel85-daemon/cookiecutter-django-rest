# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter scaffolding template that generates production-ready Django REST Framework APIs with batteries included. The next milestone is completing the modernization initiative: ensuring the template targets Python 3.13+, Django 5.0+, and PostgreSQL 16.4+, with updated dependencies, CI, and generated project structure reflecting current best practices. Daemon should focus on executing the tasks in `.daemon/specs/` — particularly the modernization and dependency update specs — validating that both the template source and its rendered output are correct, functional, and fully tested.
