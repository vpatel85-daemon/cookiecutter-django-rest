# Goal

We are building cookiecutter-django-rest, a Cookiecutter template that generates production-ready Django REST Framework APIs with auth, user management, Docker, and CI already wired up. The next milestone is to modernize the template to fully leverage Python 3.13 and Django 5.0+ features, ensure all generated project dependencies are up to date, and improve the out-of-the-box developer experience. Daemon should focus on dependency hygiene, template quality improvements, and expanding test coverage within the generated project scaffold.
