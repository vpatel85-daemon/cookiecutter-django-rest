# Goal

We are building and maintaining cookiecutter-django-rest, a scaffolding template that generates production-ready Django REST Framework APIs with best practices baked in. The next milestone is keeping the template current — updating dependencies to their latest stable versions (Python 3.13+, Django 5.0+, PostgreSQL 16.4+, DRF latest), modernizing any outdated patterns in the generated project scaffold, and ensuring CI passes cleanly. Daemon should focus on dependency updates, modernization of the template internals, and expanding test coverage for the generated project's user app.
