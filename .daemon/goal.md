# Goal

We are building and maintaining cookiecutter-django-rest, a best-practice scaffolding template for production-ready Django REST Framework APIs. The next milestone is keeping the template current with the latest stable versions of Python, Django, DRF, and PostgreSQL, while expanding the generated project's feature coverage (e.g., improved auth flows, richer test fixtures, and updated MkDocs configuration). Daemon should focus on dependency modernization, template quality improvements, and ensuring the CI pipeline reliably validates generated projects.
