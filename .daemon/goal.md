# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter template that scaffolds production-ready Django REST Framework APIs with authentication, user management, Docker, and CI baked in. The next milestone is completing the modernization effort tracked in `.daemon/specs/cookiecutter-django-rest-modernization/` — updating the template to Python 3.13+, Django 5.0+, and PostgreSQL 16.4+, ensuring all generated project code, dependencies, and CI pipelines reflect current best practices. Daemon should focus on working through the open tasks in the active specs, keeping the template lean and the generated output fully functional and well-tested.
