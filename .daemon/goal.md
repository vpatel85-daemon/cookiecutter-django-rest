# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter template that scaffolds production-ready Django REST Framework APIs with authentication, user management, Docker, and CI pre-configured. The next milestone is completing the modernization effort tracked in `.daemon/specs/cookiecutter-django-rest-modernization/` — upgrading all template dependencies to their latest stable versions, adopting current Django and DRF best practices, and ensuring the generated project reflects the Python 3.13+ / Django 5.0+ ecosystem. Daemon should focus on working through the open tasks in `.daemon/specs/`, keeping the template's generated output clean, well-tested, and aligned with the spec documents already in place.
