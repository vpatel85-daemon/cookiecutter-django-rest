# Goal

We are building cookiecutter-django-rest, a best-practice cookiecutter template that scaffolds production-ready Django REST API projects with authentication, user management, Docker support, and full test coverage built in. The next milestone is to modernize the template to fully support Python 3.13 and Django 5.0+, ensuring all generated project dependencies, configs, and CI workflows reflect current best practices. Daemon should focus on auditing and updating the template files — including settings, dependencies, and GitHub Actions workflows — to eliminate outdated patterns and ensure the generated projects pass all tests cleanly out of the box.
