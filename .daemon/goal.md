# Goal

We are building and maintaining cookiecutter-django-rest, a Cookiecutter template that lets developers scaffold production-ready Django REST Framework APIs in seconds. The next milestone is completing the modernization of the template to Python 3.13+ and Django 5.0+, ensuring all generated project dependencies, Docker images, and CI workflows reflect current best practices. Daemon should focus on executing the open modernization specs in `.daemon/specs/`, updating dependencies, and keeping the generated project's patterns consistent and well-tested.
